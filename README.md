# Netflix Login & Register UI

Project ini adalah tampilan Login dan Register menggunakan React, React Hook Form, Zod, dan Tailwind CSS dengan tema Netflix.

## ✨ Fitur
- Login Form
- Register Form
- Validasi input (Zod)
- UI modern (Netflix style)
